# Screenshot Checklist

## User Screens

- Register/Login screen
- User dashboard
- File upload success message
- My Files table
- Metadata panel
- Download action
- Shared With Me table
- Expiring link creation
- Logout

## Admin Screens

- Admin dashboard
- Users list
- Files list
- Access logs table

## Azure Screens

- App Service running overview
- GitHub Actions successful run
- Blob container with encrypted `.enc` file
- PostgreSQL resource overview
- Key Vault overview

## Security/Test Screens

- Unauthorized access denied
- Expired link error
- Integrity check failure, if prepared safely
- Rate limit or invalid upload error

## Redaction Rules

Do not show:

- connection strings
- storage account keys
- database passwords
- JWT tokens
- Key Vault secret values
- real personal data
